# Migrations

Whether you're moving from another panel, or just changing how your existing Calagopus install is deployed, pick the guide that matches what you're doing:

::::tabs
=== Pterodactyl
Coming from Pterodactyl? See the [Migrating from Pterodactyl](./pterodactyl.md) guide.

=== Pelican
Coming from Pelican? See the [Migrating from Pelican](./pelican.md) guide.

=== Docker
Already running Calagopus standalone, and want to move it to Docker? See the [Migrating to Docker Compose](./calagopus/docker.md) guide.

=== Standalone
Already running Calagopus in Docker, and want to move it to a standalone install? See the [Migrating to Standalone](./calagopus/standalone.md) guide.
::::
