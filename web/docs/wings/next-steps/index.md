# Next Steps

Wings is installed and connected to the panel. At this point Wings is running, but it doesn't do anything useful yet, since the panel needs somewhere on this node to actually place servers. That's what's next.

## Set Up Allocations

Allocations are the IP and port combinations the panel assigns to game servers so players can actually connect to them. Without at least one allocation set up, you won't be able to create a server on this node. See [Setting up Allocations](./setting-up-allocations.md) to create them.
