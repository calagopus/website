# Setting up OAuth2

This guide explains how to link OAuth2 providers to your Calagopus Panel. Using OAuth2 allows users to login to your panel without having to manually create an account.

### GitHub OAuth

Please see the [GitHub OAuth](./oauth2/github.md) documentation in the OAuth2 part of the documentation.

### Google OAuth

Please see the [Google OAuth](./oauth2/google.md) documentation in the OAuth2 part of the documentation.

### Discord OAuth

Please see the [Discord OAuth](./oauth2/discord.md) documentation in the OAuth2 part of the documentation.

### Self-hosted (authentik, Pocket-ID, authelia, etc.)

Please see the [Self-hosted OAuth](./oauth2/selfhosted.md) documentation in the OAuth2 part of the documentation.
