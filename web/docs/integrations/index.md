# Integrations
Calagopus integrates with third-party billing platforms and developer tools. Pick whichever applies to you:

::::tabs
=== VS Code
Edit server files and manage your panel directly from your editor.

See the [VS Code](./vscode.md) guide.

=== Paymenter
Automate server provisioning through Paymenter.

See the [Paymenter](./paymenter.md) guide.

=== WHMCS
Automate server provisioning through WHMCS.

See the [WHMCS](./whmcs.md) guide.

=== Blesta
Automate server provisioning through Blesta.

See the [Blesta](./blesta.md) guide.
::::
